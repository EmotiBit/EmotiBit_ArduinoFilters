20  * @brief Create a new ShiftRegisterOut object with a shift register

23  *
24  * Multiple shift registers can be cascaded by connecting the serial output
25  * of the first one to the input of the second one:
26  * ```
27  * clockPin >───────────┬──────────────────────┬───────── ⋯
28  * ┏━━━━━━━┷━━━━━━━┓ ┏━━━━━━━┷━━━━━━━┓
29  * ┃ SH_CP ┃ ┃ SH_CP ┃
30  * dataPin >───┨ DS Q7S ┠──────┨ DS Q7S ┠─ ⋯
31  * ┃ ST_CP ┃ ┃ ST_CP ┃
32  * ┗━━━━━━━┯━━━━━━━┛ ┗━━━━━━━┯━━━━━━━┛
33  * latchPin >───────────┴──────────────────────┴───────── ⋯
34  * ```
35  *
36  * @param dataPin
37  * The digital output pin connected to the serial data input (DS or
38  * SER) of the shift register.
39  * @param clockPin

41  * SRCLK) of the shift register.
42  * @param latchPin
43  * The digital output pin connected to the latch pin (ST_CP or
44  * RCLK) of the shift register.

48  */
